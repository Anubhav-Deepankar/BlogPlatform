<h1 align="center">📝 Blog Platform</h1>

<p align="center">
A full-stack blog application where users can create, edit, view, and delete blog posts.
The project is built using <b>Node.js, Express, and EJS</b>.
</p>

<p align="center">
<b>🚀 Live Demo:</b><br>
<a href="https://blogplatform-7wzu.onrender.com/" target="_blank">
https://blogplatform-7wzu.onrender.com/
</a>
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
<li>Create new blog posts</li>
<li>View all blog posts</li>
<li>Edit existing blog posts</li>
<li>Delete blog posts</li>
<li>Server-side rendering using EJS templates</li>
<li>Clean and simple UI</li>
</ul>

<hr>

<h2>🛠️ Tech Stack</h2>

<ul>
<li><b>Node.js</b> – Backend runtime</li>
<li><b>Express.js</b> – Server framework</li>
<li><b>EJS</b> – Template engine for server-side rendering</li>
<li><b>HTML & CSS</b> – Frontend structure and styling</li>
<li><b>JavaScript</b> – Application logic</li>
</ul>

<h2>⚙️ Installation & Setup</h2>

<h3>1. Clone the repository</h3>

<pre>
git clone https://github.com/Anubhav-Deepankar/BlogPlatform.git
</pre>

<h3>2. Navigate to the project directory</h3>

<pre>
cd BlogPlatform
</pre>

<h3>3. Install dependencies</h3>

<pre>
npm install
</pre>

<h3>4. Run the server</h3>

<pre>
node index.js
</pre>

<p>The server will start and the blog platform will run locally.</p>

<hr>

<h2>💡 How It Works</h2>

<ul>
<li>The backend is built using <b>Node.js</b> and <b>Express</b>.</li>
<li><b>EJS templates</b> render dynamic blog pages.</li>
<li>Users can create, view, edit, and delete blog posts.</li>
<li>Routes handle CRUD operations for blog content.</li>
</ul>
